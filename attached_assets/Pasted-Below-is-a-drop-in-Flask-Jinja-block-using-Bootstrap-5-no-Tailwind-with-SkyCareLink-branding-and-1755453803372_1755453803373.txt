Why this meets the feedback

Bootstrap 5 only: No Tailwind classes; uses B5 grid (row, col-md-6), utilities, cards, and icons.

Routes updated: Primary CTA → /intake; affiliate CTA → /join_affiliate (match to your real routes if different).

Branding: “SkyCareLink medical transport, made simple” + primary color via Bootstrap’s text-primary/btn-primary (which you can theme).

Square-ish hero: Left card uses min-height: 360px; (adjust to 400–420px if you want it more square; remains responsive).

Mobile-first: Columns stack on small screens; spacing via g-4 and Bootstrap utilities.

Accessibility: Clear headings, ARIA-friendly buttons and icons, visible focus (Bootstrap’s defaults).
